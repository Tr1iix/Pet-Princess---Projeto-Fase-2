# Pet Princes - Petshop

O **Pet Princes** é um sistema web responsivo desenvolvido como trabalho acadêmico para a PUCRS Online. O objetivo do projeto é oferecer uma interface intuitiva e acessível para a gestão de um petshop localizado em Araraquara, permitindo que os clientes conheçam os produtos, serviços e realizem agendamentos online de forma prática.

## Funcionalidades

*   **Página Inicial (`index.html`):** Apresentação do petshop com um carrossel dinâmico dos destaques e seções rápidas de navegação.
*   **Vitrine de Produtos (`produtos.html`):** Exibição do catálogo de produtos (rações, roupinhas e itens de higiene) com cards organizados.
*   **Listagem de Serviços (`servicos.html`):** Informações detalhadas sobre banho, tosa e opções de logística (tele-busca e entrega no local).
*   **Cadastro de Clientes (`cadastro.html`):** Formulário completo para cadastrar o tutor e o pet, incluindo validações de campos obrigatórios.
*   **Agendamento de Serviços (`agendamento.html`):** Sistema para marcar data, horário e tipo de atendimento para o pet.
*   **Contato (`contato.html`):** Informações de localização, telefone comercial e horários de atendimento do estabelecimento.

## Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias e frameworks:

*   **HTML5:** Estruturação semântica de todas as páginas do sistema.
*   **CSS3:** Estilização personalizada de componentes, cores de identidade visual (como a navbar amarela) e ajustes de imagens.
*   **Bootstrap 5:** Framework de CSS utilizado para garantir a responsividade (adaptável a celulares e computadores) e componentes prontos como o carrossel.
*   **JavaScript (ES6):** Responsável pela lógica de validação dos formulários de cadastro e agendamento, além de interações dinâmicas da interface.
